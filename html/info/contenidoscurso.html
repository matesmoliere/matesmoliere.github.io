Contenidos del Curso

Material para el desarrollo del curso:

  • Cuaderno de apuntes obligatorio tanto en las Prácticas, como en la Teoría.
  • Página web del curso

Primer Trimestre:

Práctica:

  • Sistema Operativo Linux
  • Trabajo desde un Terminal
  • Estructura de archivos
  • Manejo de archivos: crear, modificar, borrar, copiar, etc.
  • Permisos: asignación y modificación
  • Órdenes más comunes: ls, mkdir, cd, etc.
  • Uso de tuberías (Pipes)
  • Búsqueda de patrones
  • Uso de los editores Gedit,Kate y Vim
  • Configuración del Entorno de trabajo personal
  • Ofimática: Uso de las herramientas de Libreoffice; Write y Calc

Teoría:
  • Historia de la Informática
  • Partes de un ordenador
  • Sistemas de almacenamiento digital
  • Códigos: binario, decimal y hexadecimal. Operaciones de conversión
  • Sistemas operativos: Linux, Windows, OS, Android

Trabajo para el 1er trimestre:
  • Creación de cuentas personales en Gmail y Dropbox
  • Inteligencia Artificial: "Desde sus principios hasta las máquinas de destrucción masiva"


Segundo Trimestre:

Práctica:

  • Editor de imágenes GIMP: manipulación de archivos de imagen
  • Editor de audio Audacity: manipulación de archivos de audio
  • Lenguaje HTML y páginas web: desarrollo de páginas HTML y CSS
  • Github y alojamiento de páginas web

Teoría:
  • Hardware necesario para multimedia
  • Características de la imagen y el vídeo digitales
  • Características del sonido digital
  • Redes de ordenadores: cableadas e inalámbricas
  • Seguridad informática
  • La placa base y los buses

Trabajo para el 2do trimestre:
  • Internet y las nuevas tecnologías digitales: Pasado y Futuro
  • Desarrollo de tu propia página web


Tercer Trimestre:

Práctica:

  • Lenguaje Logo y Kturtle
  • Lenguaje de programación Python

Teoría:
  • Lenguajes de Programación:
    • Algoritmos
    • Definición
    • Historia
    • Elementos de un programa
    • Implementación
    • Paradigmas
  • Protocolos de Internet: TCP/IP; DNS, etc.
  • Lógica Boolena y puertas lógicas

Trabajo para el 3er trimestre:

Crea tu propio juego en Python. Dos opciones:
  • Juego gráfico: el personaje puede moverse con el teclado
  • Preguntas y respuestas (p.e. la Mazmorra)
